# Shelly Microservice
This Microservice provides an integration for Shelly devices including switching them on/off, configuring them and showing the consumption of the plugged in devices.
A Shelly is designed for smart homes for power conservation and automation.

# Usage

### Install all necessary packages
```bash
npm install
```

### Start the Service
```bash
node server.js
```

### Test the Service
```bash
npm test -- --coverage
```
